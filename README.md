# 个人资产钱包 PWA

一个 iPhone 优先的轻量个人资产余额管理应用。应用只记录各账户当前余额，并在余额发生变化时自动写入余额历史，用于计算总资产、总负债、净资产和趋势图。

## 运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物会输出到 `dist/`，包含 PWA manifest 和 service worker，可部署到静态站点服务。

## 已实现

- 首页展示净资产、总资产、总负债
- 按分组展示资产和负债账户
- 预设账户类型初始化
- 新增、编辑、软删除账户
- 点击账户后修改余额，余额变化自动记录历史
- 净资产趋势、总资产/总负债趋势、单账户余额趋势
- JSON 导入导出、余额历史 CSV 导出、清空数据
- 隐藏金额、深色模式、小数位设置、显示已删除账户
- IndexedDB + Dexie 本地存储
- iPhone 安全区适配和 PWA 图标
